# PBRTQC 


This repository contains code for a simulation-based study comparing patient-based real-time quality control (PBRTQC) with machine learning models for detecting analytical and pre-analytical errors in clinical chemistry data.

## Contents
- Simulated data generation
- PBRTQC analysis
- Machine learning models: logistic regression, random forest, XGBoost
- Results table and figure generation

## Notes
This repository includes code only. Data are simulated within the scripts.
